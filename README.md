# KolSport AI

## Personal AI Coach for Endurance Athletes

KolSport AI is a personal AI coach designed to analyze long-term athlete data.

The goal is not to replace a coach, but to help make better training decisions using data from Garmin, laboratory tests, lactate measurements, training history and subjective feelings.
